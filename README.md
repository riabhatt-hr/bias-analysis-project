# Bias Analysis Project

##  Overview
This project focuses on analyzing bias in datasets using structured data analysis techniques. The goal is to identify patterns, inconsistencies, and potential biases in decision-making data.

The dataset used in this project contains structured information that has been cleaned, processed, and analyzed to extract meaningful insights.

---

##  Objectives
- Identify bias in data
- Analyze patterns across different variables
- Compare outcomes across categories
- Support fair and data-driven decision-making

---

##  Dataset
- File: `bias_analysis_final.xlsx`
- Type: Structured dataset (Excel format)
- Contains variables related to decision outcomes, categories, and evaluation metrics

---

##  Tools & Technologies Used
- Microsoft Excel
- Data Cleaning & Preprocessing
- Basic Statistical Analysis
- Visualization (if applied)

---

##  Key Analysis Performed
- Data cleaning and formatting
- Categorization of variables
- Comparative analysis
- Identification of biased trends (if any)

---

##  Insights
- Highlighted areas where bias may exist
- Identified patterns across different groups
- Provided data-backed observations

##  How to Use

3. Review sheets and analysis performed
